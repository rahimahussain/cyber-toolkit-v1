# cyber-toolkit-v1
A beginner cybersecurity project built in Python to learn the basics of network reconnaissance and security tools.

## Features
- Port Scanner (detects open ports on a target system)
- Network Info Tool (shows local system information)
- File Hash Checker (SHA-256 file integrity verification)

## Skills Learned
- TCP/IP networking fundamentals
- Basic socket programming in Python
- Port scanning concepts
- Intro to cybersecurity reconnaissance

## Purpose
This project is part of my journey into cybersecurity, focusing on building practical tools to understand how attackers and defenders think.

## Disclaimer
For educational use only in safe environments.
